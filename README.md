## Описание

Веб-приложение предназначенное для управления балансами игроков, сгруппированных по игровым девайсам. Приложение получает данные с *backend*-сервера, отображает список устройств и связанных с ними игроков, а также позволяет изменять их баланс.  

## Функциональность

- Отображение списка всех девайсов, доступных через *API* - `https://dev-space.su/swagger/index.html`
- Просмотр игроков внутри выбранного устройства
- Переход на страницу конкретного игрока с отображением его текущих балансов
- Изменение баланса игрока (ввод, проверка и отправка значения на сервер)
- Валидация числового ввода:
- - разрешены только числа
- - максимальная точность - две цифры после десятичного разделителя
- - блокировка некорректных символов
- Генерация уникальных имён игроков при первом запуске:
- - backend не возвращает имён - только `id`
- - при первой загрузке приложения, каждому игроку создаётся случайное имя и фамилия
- - имена привязываются к `deviceId` и `id`
- - данные сохраняются в **LocalStorage** 
- - при последующих загрузках имена подставляются автоматически
- Обработка перехода по ссылке на несуществующего игрока - возврат на главную страницу
- Строгая типизация всей клиентской части

## Запуск приложения

Для запуска приложения необходим:
- `nodejs >=20.19.5` 
- `npm >=10.8.2`

```
npm i
npm run dev
```
